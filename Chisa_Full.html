Quick presets
Tap a preset — file will download to your Downloads folder. Then open FV File Explorer → Downloads → Copy → Paste → Overwrite ClientGraphicsConfig.json.
Tip: After applying, restart the game if you see no change.
Advanced tips
If you want maximum sharpness with less blur, use RenderScale 110–125% and SharpenIntensity 0.45–0.55. If device gets hot, switch to BatterySaver or AntiStutter.
